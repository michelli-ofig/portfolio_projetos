Projetos em Ciência de Dados

Construção de um programa de fidelidade com clusterização de clientes

Nesse projeto utilizei Python, Estatística e técnicas não-supervisionadas de Machine Learning para segmentar um grupo de clientes com base em suas características de performance de compra, a fim de selecionar grupos de clientes para formar um programa de Fidelidade com o objetivo de aumentar a receita da empresa. E o resultado dessa solução, caso fosse implementada, seria de R$ 15MM de dólares de receita anual.

  • Python, Pandas, Matplotlib e Seaborn;

Predição de vendas para as próximas 6 semanas de uma rede com mais de 3000 drogarias

Neste projeto foi utilizado Python, Estatística e algoritmos de Regressão para prever as vendas diárias com até 6 semanas de antecedência, com o objetivo de alocar recursos para a reforma das lojas. As vendas são influenciadas por muitos fatores, incluindo promoções, competição, feriados escolares e estaduais, sazonalidade e localidade. Os algoritmos Random Forest Regressor e XGBoost Regressor trouxeram resultados de quase 90% de assertividade para o modelo.

Ferramentas Utilizadas:

  • Git, Gitlab e Github;
  • Python, Pandas, Matplotlib e Seaborn;
  • Jupyter Notebook;
  • Algoritmos de Machine Learning: Average Model, Linear Regression, Linear Regression Regularized, Random Forest Regressor e XGBoost Regressor;
  • Hyperparameter Fine Tuning: Random Search.
